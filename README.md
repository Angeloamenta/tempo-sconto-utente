# Tempo Sconto Utente – WordPress Plugin

## 🇮🇹 Descrizione

**Tempo Sconto Utente** è un plugin per WooCommerce che applica automaticamente uno sconto temporaneo personalizzato a un singolo prodotto per ogni utente.

🕒 Il timer parte quando l'utente visita una pagina contenente lo shortcode.

🎯 Ideale per creare offerte a tempo limitato visibili solo al singolo utente.

### ✅ Funzionalità
- Sconto valido solo per l'utente attivo
- Timer configurabile (es. 30 minuti)
- Prezzo scontato applicato come voce separata nel carrello
- Shortcode disponibili:
  - `[tsu_timer]` → Avvia e mostra il timer
  - `[tsu_timer_display]` → Mostra solo il countdown (senza farlo partire)
  - `[tsu_prezzo]` → Mostra prezzo originale e scontato
- Pannello di controllo nel backend per gestire:
  - Prodotto da scontare
  - Durata del timer
  - Prezzo scontato
  - Reset del timer per tutti gli utenti

## 🇬🇧 Description

**Tempo Sconto Utente** is a WooCommerce plugin that applies a temporary discount to a specific product, individually for each user.

🕒 The countdown starts when the user opens a page containing the shortcode.

🎯 Perfect for time-sensitive offers targeted to individual visitors.

### ✅ Features
- Discount applies only to the active user
- Configurable countdown (e.g., 30 minutes)
- Discount shown as a separate fee in the cart
- Available shortcodes:
  - `[tsu_timer]` → Starts and displays countdown
  - `[tsu_timer_display]` → Displays countdown only
  - `[tsu_prezzo]` → Shows original and discounted price
- Admin panel:
  - Select discounted product
  - Set countdown duration
  - Define discounted price
  - Reset timers for all users

---

## 📦 Installazione / Installation

1. Scarica il file ZIP oppure clona questo repository
2. Carica la cartella nella directory `/wp-content/plugins/`
3. Attiva il plugin da **Plugin > Plugin installati**
4. Vai su **Tempo Sconto** per configurare

---

## 🔖 Requisiti / Requirements
- WordPress 5.0+
- WooCommerce attivo
- Permessi da amministratore per configurarlo
