Applica uno sconto personale e temporaneo a un prodotto WooCommerce selezionato, con timer visuale e shortcode personalizzabili.

== Description ==

Tempo Sconto Utente permette di impostare uno sconto temporaneo e personale su un prodotto WooCommerce. 
L’utente vede un timer countdown che parte quando visita una pagina con lo shortcode e ha X minuti per acquistare al prezzo scontato. 

Caratteristiche:
* Selezione del prodotto da scontare nel backend
* Impostazione durata timer in minuti
* Inserimento prezzo scontato
* Shortcode per timer, prezzo originale, prezzo scontato e visualizzazioni personalizzate
* Sconto applicato globalmente durante la durata del timer, anche nel carrello e checkout
* Facile da usare e personalizzare

== Installation ==

1. Carica la cartella `tempo-sconto-utente` nella directory `wp-content/plugins/`.
2. Attiva il plugin attraverso il menu “Plugin” in WordPress.
3. Vai su “Tempo Sconto” nel menu di amministrazione per configurare prodotto, durata e prezzo.
4. Usa gli shortcode nelle pagine o nei post per mostrare timer e prezzi.

== Frequently Asked Questions ==

= Posso applicare lo sconto a più prodotti? =  
Al momento il plugin supporta solo un prodotto scontato per volta.

= Lo sconto si applica automaticamente nel carrello? =  
Sì, se il timer è attivo e l’utente ha visitato la pagina con lo shortcode timer.

= Posso resettare il timer per tutti gli utenti? =  
Sì, dal backend puoi resettare il timer per far partire una nuova offerta.
