== Description ==
Questo plugin consente di applicare **uno sconto temporaneo e personale** a un prodotto WooCommerce, attivato da un timer personalizzato per ogni utente.

== Features ==
* Timer visibile solo all'utente che visita una specifica pagina
* Prezzo scontato solo per l'utente con timer attivo
* Sconto visibile ovunque: pagina prodotto, shop, carrello
* Backend intuitivo per configurare prodotto, durata e prezzo
* Shortcode facili da usare per timer e prezzi

== Shortcodes ==
- `[tsu_timer]`: Avvia il timer per l'utente
- `[tsu_timer_display]`: Mostra il countdown (se attivo)
- `[tsu_prezzo]`: Mostra prezzo originale + scontato
- `[tsu_sconto_display]`: Mostra solo il prezzo scontato
- `[tsu_prezzo_originale]`: Mostra solo il prezzo originale

== Installation ==

1. Attiva il plugin
2. Vai su **Tempo Sconto** nel backend per configurare
3. Usa gli shortcode nelle pagine
